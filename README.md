# 🚀 Projet : Entraide & Évaluation Collective

> "Seul on va plus vite, ensemble on va plus loin."

---

## 💡 Le Concept
Bienvenue sur ce dépôt ! L'objectif est simple : transformer notre évaluation individuelle en une **opportunité d'apprentissage collaboratif**. Plutôt que de travailler en vase clos, nous centralisons nos travaux ici pour progresser ensemble.

> [!IMPORTANT]
> **L'idée n'est pas de copier-coller**, mais de mieux comprendre en observant différentes approches d'un même problème.

---

## 🛠 Comment ça marche ?

| Action | Description |
| :--- | :--- |
| **Chacun sa branche** | Chaque participant crée sa propre branche (ex: `nom-prenom`) pour pousser son code. |
| **Transparence** | Navigue librement entre les branches pour voir comment les autres ont structuré leur projet. |
| **Entraide** | Un blocage sur une logique ou une syntaxe ? Regarde les solutions des copains ou ouvre une **Issue** ! |

---

## ⚠️ Règles de Sécurité & Workflow

* **🚫 Interdiction de Push** : Il est formellement interdit de `push` sur une branche qui ne vous appartient pas. Respectez le travail et l'espace de chacun.
* **🛡 Sécurité Personnelle** : Par mesure de précaution (et pour vos archives personnelles), nous vous encourageons vivement à **maintenir un second dépôt Git privé** de votre côté pour sauvegarder votre progression régulièrement.

---

## 🤝 Règles de bonne conduite

* **✨ Respect du travail** : On s'inspire, on ne pille pas sans comprendre.
* **💬 Bienveillance** : Les commentaires et les retours sur le code des autres doivent rester constructifs.
* **📚 Partage** : Si tu as trouvé une astuce géniale, n'hésite pas à la documenter directement sur ta branche.

---
